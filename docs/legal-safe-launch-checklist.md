# Legal-Safe Launch Content Checklist — India Interest Radio (Safe-First MVP)

**Scope:** Kids' stories + Moral stories + Devotion + Astrology
**Languages:** English, Hindi, Kannada, Marathi, Telugu, Tamil, Bengali (+ expand later)
**Core rule:** Every audio item MUST fall into ONE of the four "safe zones" below. If it doesn't, do not ship it.

---

## 0. The 4 Safe Zones (memorize these)

| Zone | Usable for ads + subscription + AI narration? | Required action |
|---|---|---|
| Public Domain (author died ≥60 yrs ago in India) | YES — original text only | Record source + death year proof |
| CC BY / CC BY-SA / CC0 | YES (commercial + derivatives allowed) | Store attribution string |
| Own / commissioned content | YES (full rights) | Keep signed work-for-hire contract |
| Licensed from owner | YES (per contract) | Keep license document |

**Forbidden (will cause legal liability):**
- [ ] NO CC-NC (NonCommercial) content — we monetize.
- [ ] NO CC-ND (NoDerivatives) content — TTS narration is a derivative.
- [ ] NO modern translations of public-domain texts (translation is freshly copyrighted).
- [ ] NO existing recordings/renditions (use only the underlying text; we generate our own audio).
- [ ] NO scraped blog/Wattpad/Pratilipi/horoscope-site text.

---

## 1. Kids' Stories — Difficulty: EASY

**Approved sources:**
- [ ] Pratham Books StoryWeaver (storyweaver.org.in) — license: **CC BY 4.0**
- [ ] Global Digital Library / African Storybook — CC BY (filter Indian languages)
- [ ] Panchatantra / Hitopadesha / Jataka Tales — **public domain** (original); retell in own words
- [ ] Aesop's Fables — public domain

**Per-item checklist:**
- [ ] Confirm license is CC BY / BY-SA / CC0 (NOT NC, NOT ND)
- [ ] Capture attribution: `Original story from Pratham Books' StoryWeaver, licensed under CC BY 4.0. [Title] by [Author]. Changes: narrated as audio.`
- [ ] Note if text was modified (translation/adaptation) → label as derivative
- [ ] Store source URL + license URL in catalog metadata

## 2. Moral Stories — Difficulty: EASY

**Approved sources:**
- [ ] Panchatantra / Hitopadesha / Jataka / Tenali Rama / Akbar-Birbal classics — **public domain** (retell in own words)
- [ ] Premchand stories (Hindi/Urdu, d.1936) — **public domain in India since 1997**
- [ ] Tagore stories (Bengali, d.1941) — **public domain in India since 2002**
- [ ] StoryWeaver moral-themed titles — CC BY 4.0

**Per-item checklist:**
- [ ] If using a named author → verify death year ≥60 years ago (India life+60 rule)
- [ ] If using a modern edition → use ONLY the original text, never a modern translation/edition's wording
- [ ] Capture attribution + source

## 3. Devotion — Difficulty: MEDIUM

**Approved sources (original ancient texts only):**
- [ ] Bhagavad Gita, Vedas, Ramayana, Mahabharata, Upanishads (original Sanskrit) — public domain
- [ ] Quran, Bible, Guru Granth Sahib (original) — public domain
- [ ] Public-domain bhajan/sloka **text** — public domain

**Per-item checklist:**
- [ ] Use ONLY the original/ancient text — NOT a copyrighted modern translation
- [ ] Generate OUR OWN narration/AI voice — do NOT reuse any existing recording (T-Series, Saregama, YouTube, etc.)
- [ ] Cultural/pronunciation QA by a native reviewer per language (devotion tone is sensitive)
- [ ] If a translation is needed → commission our own or use a CC BY translation only

**Hard NO:**
- [ ] Any commercial devotional track, artist rendition, or discourse recording.

## 4. Astrology — Difficulty: MEDIUM (compute, never copy)

**Approved method:**
- [ ] Compute planetary positions ourselves — facts are not copyrightable.
- [ ] Ephemeris engine: **Skyfield (MIT license)** + **NASA JPL DE ephemeris (public domain)** → free + closed-source OK.
  - [ ] AVOID Swiss Ephemeris unless we buy the commercial license (~700 CHF) — its AGPL forces open-sourcing the whole app.
- [ ] Generate horoscope TEXT ourselves (own templates or LLM) — never scrape horoscope sites.
- [ ] Generate audio via our TTS per sign per language, daily.

**Per-item checklist:**
- [ ] Confirm ephemeris library license = permissive (MIT/BSD/Apache) or PD data
- [ ] Horoscope text is generated by us (log the generation method)
- [ ] Daily job produces 12 signs x N languages, then TTS

---

## 5. Cross-Cutting Compliance (Do NOT skip)

- [ ] **Attribution registry:** every audio item has source, author, license, license URL, and "changes made" stored in catalog metadata.
- [ ] **Privacy policy + Terms** published (mandatory for Play Store + storing user data).
- [ ] **User data stored = profile + preferences + favorites + consent ONLY.** No audio relayed/hosted from third parties.
- [ ] **Takedown process:** a contact + workflow to remove any item on valid complaint.
- [ ] **UGC (if enabled later):** ToS must grant us a license to narrate; add reporting + moderation.
- [ ] **TTS voice license check:** confirm the TTS provider's terms allow commercial distribution of generated audio (Google/Azure/Polly: yes; verify per plan).
- [ ] **Per-language voice QA:** sign off audio quality per language before that language goes live (esp. devotion + regional languages).

---

## 6. Launch Gate (don't publish until all true)

- [ ] 100% of MVP catalog items mapped to a safe zone with stored attribution.
- [ ] Zero NC / ND / scraped / modern-translation / existing-recording items.
- [ ] Ephemeris + TTS provider licenses verified for commercial use.
- [ ] Privacy policy, terms, takedown contact live.
- [ ] Native-speaker QA passed for each launched language.
